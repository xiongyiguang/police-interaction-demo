# police-interaction-demo

`所长直通车` 原型演示页面仓库，包含群众入口、接访工作台、工单中心、知识库管理、治理看板等多个本地 HTML 页面。

## 页面概览

- `1.群众侧｜企业微信对话窗口（入口界面）1.html`
- `1.群众侧｜企业微信对话窗口（入口界面）2.html`
- `1.-群众互动-安全诉求智能核实与紧急接入.html`
- `1-.群众互动-噪音纠纷智能预处与人工介入.html`
- `2.接访侧｜会话列表与队列工作台（接 + 分 的主界面）.html`
- `3.接访侧｜会话处理界面（智能辅助 + 人工接续）.html`
- `4.接访侧｜智能补充信息采集页（否→补充处理→回主线的关键页）.html`
- `5.智能侧｜知识库管理界面（知识条目口径版本）.html`
- `6.办理与归档侧｜接访工单与小结界面（办 + 记 的核心）.html`
- `7.管理侧｜运行看板与治理分析界面（管 的核心）.html`

## 业务链路

`群众入口 -> 智能预处 -> 接访工作台 -> 信息补充 -> 工单中心 -> 治理看板`
